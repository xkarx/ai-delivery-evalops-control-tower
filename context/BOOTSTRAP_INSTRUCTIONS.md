# How to Start in Codex

1. Create a new private GitHub repository for the control tower.
2. Upload this pack to the repository root.
3. Connect the repository to Codex.
4. Create a Codex cloud environment.
5. Start a new Codex task and paste `codex/INITIAL_PROMPT.md`.
6. Review Phase 00 changes.
7. Continue with the orchestrator or phase prompts.
8. Add credentials only after adapters and health checks exist.

Do not paste the full brainstorming conversation. The repository is the durable source of truth.
