# Workflows

## Evidence to approved feature

Ingest → retrieve → cluster → brainstorm → score → recommend → human approve/reject → decision log → PRD → metrics.

## Approved feature to PR

Create tickets → map dependencies → inspect repo → isolate work → implement → test → open PR → link lineage.

## Eval and release

Select dataset/rubric → deterministic graders → semantic graders → human review where required → aggregate → pass/block/escalate → preview → approval → deployment.

## Production feedback

Ingest events/errors → detect issue → capture trace → classify → create regression case → create follow-up ticket → rerun campaign → close incident.
