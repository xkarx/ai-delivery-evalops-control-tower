# Integrations

## GitHub

Repository selection, issues, branches, PRs, checks, workflows, deployment/release links, and webhooks.

## Slack

Start workflow, status, questions, approval/rejection actions, alerts, incidents, and deep links.

## Linear

Projects, issues, dependencies, status synchronization, and deep links. Provide GitHub Issues fallback.

## Supabase

Migrations, seed scripts, lineage data, security plan, and health checks.

## Langfuse

Nested traces, generations, retrieval, tool calls, scores, dataset/experiment links, cost, latency, and trace URLs.

## PostHog

Persistent distinct IDs, event capture, funnels, feature exposure, adoption, segments, experiments, and dashboard links.

## Deployment

Preview URL, environment, commit SHA, feature ID, status, and release record.
